No No Zone

No No Zone is a simple way to block access to time-consuming websites ( also known as NO NO ZONES ) as you specify in the window below, or using one of the togles for the website

Type the sites you would like to block the access to in a format like facebook.com, type them in new line each. Sites will be blocked on their next attempted visit if enabled. TO OPEN THIS TAB UP AGAIN RIGHT CLICK ON THE EXTENTION AND SELECT OPTIONS. Please note that by default this extention is enabled!!
